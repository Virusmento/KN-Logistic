# KN-Logistic — Внутрішній портал (локальна збірка)

Коротко: оновлена статична версія внутрішнього порталу зі стилями, скриптом та базовими файлами манифесту.

Файли в проєкті:
- `index.html` — оновлена сторінка
- `styles.css` — основні стилі
- `script.js` — поведение меню та обробка форми
- `assets/logo.svg`, `assets/favicon.svg` — прості іконки
- `manifest.json` — PWA-манифест
- `robots.txt` — заборона індексації

Запуск локально:
Відкрийте `index.html` у браузері (подвійний клік або через `Live Server`).

Примітки:
- Сайт призначений для внутрішнього використання. `robots.txt` і мета-теги попереджають пошуковики.
- Форма на сторінці працює в демо-режимі (імітація відправки).

- Налаштувати CI/CD і деплой у внутрішню мережу
